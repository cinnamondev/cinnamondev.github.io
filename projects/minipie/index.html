Pie handheld

Handheld device powered by the Compute Module 4. To be finished, parts are on backorder, with a custom shell being designed.

Github

Design

Initial idea: CM4 carrier connected to a screen in a GBA SP shell.

Issues with initial idea:

  • Thin dimensions, not much room to work with for cooling/batties/etc..
  • Smaller dimenions meant only smaller batteries (~2000mAh) could be used.
  • Relatively high power consumption of CM4 meant this may mean issues with short (<1h) battery life
Too thin to store larger batteries, smaller size worsened problem. No exact dimensions available to work with Screen was quite small.

New idea: CM4 carrier connected to a screen in a custom, 3d printed shell. Screen is on a slider mechanism to reveal gamepad/accessory pad. Magnets used to lock screen in desired positions.
