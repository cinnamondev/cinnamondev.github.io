Pi Handheld

Handheld portable designed around the Compute Module 4, which is a SBC designed for use with custom designs.

This was my first major electronics project, using KiCAD for PCB design and F360 for designing the casing, which was 3D printed.

(CW) PIC Keypad

Used MPASM assembly on a PIC16 to develop a 'code-checker' that read a keypad using multiplexer techniques that are similar to how any traditional keyboard is read today.

There was also an additional project to develop a dtmf transmitter/reciever.

Source will not be shared as this is board submitted work.
